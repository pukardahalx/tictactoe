# Classic Tic-Tac-Toe Game

A simple implementation of the classic two-player game, Tic-Tac-Toe, using HTML, CSS, and pure JavaScript.

## Live Website

You can play the game live here:
**https://pukardahalx.github.io/tictactoe/** 

---

## Features

* **Player vs Player (PvP):** Supports two local human players (X and O).
* **Game Status:** Clearly indicates whose turn it is and the final outcome (Win or Draw).
* **Simple Logic:** Clean, standard rules for game progression and win detection.

## How to Play

1.  **Start:** Player X always goes first.
2.  **Move:** Click any empty cell on the board.
3.  **Goal:** Get three of your marks (X or O) in a row (horizontal, vertical, or diagonal) to win.
4.  **Reset:** Use the "Restart Game" button to begin a new round.

## Technologies Used

* HTML5
* CSS3
* JavaScript
